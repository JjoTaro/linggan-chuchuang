# 灵感橱窗

放在手边、一直亮着的一块屏：一张安静的卡片，按设定的间隔自动轮换，跨十几个领域（宇宙、数学、认知、博弈、历史、哲学、计算、金融……）。看到喜欢的点 ★ 存进灵感库，回头一键复制成 Markdown 拿去深挖。

纯静态单文件，无需后端、无需打包，离线可用。

## 文件

| 文件 | 作用 |
| --- | --- |
| `index.html` | 应用本体（HTML + CSS + JS 全在一个文件里） |
| `manifest.webmanifest` | PWA 清单，让它可「安装」 |
| `icon-180.png` / `icon-512.png` | 应用图标 |
| `worker.js` | 可选：AI 代理脚本（让 API Key 留在服务器端） |

## 部署到 GitHub Pages

1. 新建一个仓库（设为 **Public**，否则免费账户的 Pages 无法工作），把以上文件传到根目录。
2. 仓库 **Settings → Pages**，Source 选 `Deploy from a branch`，分支选 `main`、目录 `/ (root)`，保存。
3. 等一两分钟，访问 `https://<你的用户名>.github.io/<仓库名>/` 即可。

## 加到 iPad 主屏幕（变全屏 App）

用 **Safari** 打开上面的网址 → 点底部「分享」→「添加到主屏幕」。之后从主屏幕图标进入就是全屏、无浏览器边框的样子。

设置里有「保持屏幕常亮」开关，会在页面前台时尝试阻止息屏（iPadOS 16.4+ 的 Safari 支持）。

## 内容来源

- **精选库**（默认）：约 70 张已核对的卡片，完全离线、免费、永久。
- **AI 现编**：在设置里把「内容来源」切到 AI 现编后，按你填的兴趣方向无限生成新卡片，**生成的卡片会存到本地、不断累积（上限 200 张），刷新和重开都还在**。

### AI 服务商（怎么选最省钱）

独立网页版没有免费的内置 AI 通道，必须接一个服务商。设置里有下拉选择，常见选项：

| 服务商 | 费用 | 备注 |
| --- | --- | --- |
| **智谱 GLM-4-Flash** | **免费** | 国产，性价比首选 |
| **DeepSeek** | 极便宜 | 国产，这点用量几乎可忽略；最标准的 OpenAI 兼容 |
| 通义千问 Qwen / Moonshot Kimi | 有免费额度 | 国产 |
| Google Gemini Flash | 有免费额度 | 跨境，境内需梯子 |
| OpenRouter | 含多个 `:free` 模型 | 跨境，境内需梯子 |
| Anthropic Claude | 最贵 | — |

### 调用方式：直连 还是 通过代理

- **浏览器直连**：把 Key 填进设置即可，最快。Key 只存在你这台设备的浏览器里，不进公开源码。**但大多数国产接口（DeepSeek / 智谱 / 通义…）不允许浏览器跨域，直连会失败**——它们要走代理。Gemini、OpenRouter 可以直连（需梯子）。
- **通过代理**（推荐给国产服务商）：按 `worker.js` 顶部说明部署一个 Cloudflare Worker，把网址填进「代理地址」。这样 Key 藏在服务器端，也绕过了跨域。一个 Worker 改改 `UPSTREAM_URL` 就能对接任意服务商。

> 小提示：你在中国大陆，最顺的组合通常是 **DeepSeek 或 GLM-4-Flash + 代理**。

## 数据存哪

灵感库、设置、API Key 都只存在你这台设备浏览器的 `localStorage` 里，不上传任何服务器。换设备或清理浏览器数据会清空。
